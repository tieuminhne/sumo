# 🍜 SUMO QUÁN - Restaurant Order Management

Multi-tenant restaurant order management system with Kitchen Display System (KDS).

## Features
- Order management with table numbers
- Kitchen Display System (KDS)
- Real-time sync with Supabase
- Mobile-friendly PWA
- Voice reading for orders

## Pages
- `index.html` - Main landing page
- `food.html` - Customer order app
- `kds.html` - Kitchen display
- `login.html` - Login
- `register.html` - Register

## Tech Stack
- HTML/CSS/JavaScript
- Supabase (Backend/Auth/Database)
- PWA with Service Worker
